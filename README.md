# PROJECT TITLE : 
MYSQL CODING CHALLENGE

## DESCRIPTION :
This project demonstrates various **MySQL concepts** through a set of practice-based coding challenges.  
It includes tasks such as table creation, applying constraints, joins, subqueries, functions, procedures, triggers, and transactions.  
Each topic is designed to strengthen practical SQL knowledge and real-time problem-solving skills.


## GETTING STARTED :

### DEPENDENCIES :
- MySQL Workbench or MySQL CLI  
- Windows 10 or above  
- Basic SQL knowledge

---

### INSTALLING :
1. Download or clone this repository.  
2. Open the `.sql` file in **MySQL Workbench**.  
3. Create the database and execute the SQL commands.

```sql
CREATE DATABASE City_Hospital_DB;
USE City_Hospital_DB;

EXECUTING PROGRAM :

1. Open MySQL Workbench and connect to your local server.
2.Run each query or section step by step.
3.Verify outputs using SELECT statements.

Example :
CALL GetEmployeeByID(3);
SELECT * FROM EmployeeFullInfo;

HELP :
If you face any issues while running queries:
* Make sure the database is selected using USE City_Hospital_DB;
* Drop tables before re-creating if duplicates exist.
* Execute dependent queries in the correct sequence.
